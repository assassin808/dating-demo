## Key Innovation

**Critical Events Hypothesis**: Relationship outcomes are determined by responses to sparse critical moments, not every interaction. We formalize compatibility as a reward modeling problem where simulated agent behavior predicts real-world relationship success.

## Results

- **Speed Dating**: Predicts mutual "yes" decisions from simulated first conversations

Our approach outperforms traditional similarity-based baselines.
